# QuickSort_MergeSort_Sequential_Algorithms
Program that sorts a text file of unique integers and outputs the sorted list. In effect, emulating the performance of the UNIX `sort -n` command. 

# Execution Examples
./quickmergealg -i inputFile.txt -o outputFile.txt --alg=quick

./quickmergealg -i inputFile.txt -o outputFile.txt --alg=merge
